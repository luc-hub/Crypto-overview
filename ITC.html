Perfect Secrecy and the One-Time Pad

Definition: Perfect secrecy is a cryptographic property where the ciphertext reveals no information about the plaintext, regardless of computational power.

The one-time pad is the only known encryption method achieving perfect secrecy. It requires the key to be:

  • As long as the message
  • Truly random
  • Used only once

Example: Encrypting the message "HELLO" using a key "XMCKL".

Plaintext: HELLO

Key: XMCKL

Ciphertext: ZIGGM (calculated using modular addition)

Shannon's Theorem

Shannon proved that a perfectly secure system must meet the following criteria:

  • The key must be as long as the message
  • The key must be truly random
  • The key must be kept secret and never reused

Perfect Authentication

Definition: Authentication ensures the integrity and origin of a message, preventing tampering and forgery.

Universal Hashing

Universal hashing is a method for creating hash functions that minimize the probability of collisions. It is key in designing secure authentication schemes.

Given a family of hash functions H, it is universal if for any distinct inputs x and y, the probability that h(x) = h(y) for a randomly chosen h in H is low.
